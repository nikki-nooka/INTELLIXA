# 🎓 Intellexa: Wisdom-Driven Adaptive Learning & Career Development Ecosystem (v2.0)

A comprehensive, full-stack educational and technical preparation platform designed to bridge the gap between academic theory, engineering practice, and ethical responsibility (**Dharma**). By combining dynamic AI roadmap synthesis, real-time memory visualizers, conversational podcast studios, and automated profile analytics, Intellexa provides a complete career-accelerator suite for the modern software engineer.

## 🚀 Setup & Execution Guide

Ensure the full-stack system is configured correctly using the instructions below.

### 📦 Prerequisites & Environment Setup
Define your secret environment variables in `.env` inside the root directory. NEVER expose secrets to the client.

```env
# .env (or .env.example)
GEMINI_API_KEY=your_secured_gemini_api_key_here
PORT=3000
NODE_ENV=development
```

### ⚙️ Command Sequence

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run in Development Mode:**
   ```bash
   npm run dev
   ```

3. **Build the Production Artifacts:**
   ```bash
   npm run build
   ```

4. **Launch the Self-Contained Server:**
   ```bash
   npm run start
   ```

*Note: The Express Node server will run on port `3000` and handle static asset streaming automatically in production mode, while proxying all Gemini API requests securely behind the scenes.*
